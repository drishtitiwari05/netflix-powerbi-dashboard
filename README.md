# Netflix Power BI Dashboard

This project analyzes Netflix content using Power BI.

## Features
- Total Titles, Movies, TV Shows KPIs
- Content growth over time
- Top countries by content
- Movies vs TV Shows comparison
- Most common rating analysis
- Interactive slicers (Country, Category, Rating, Release Year)

## Tools Used
- Power BI
- Power Query
- DAX
- Excel Dataset

## Insights
- Movies make up ~73% of Netflix catalog
- Content grew rapidly after 2016
- USA and India produce most Netflix titles
- TV-MA is most common rating
- Average movie runtime ≈ 102 minutes
